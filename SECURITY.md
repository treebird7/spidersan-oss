# Security Policy

## Reporting a Vulnerability

For security issues, please use GitHub's private vulnerability reporting rather than public issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Current | ✅ |
| < Current | ❌ |

## Security Practices

- API keys in environment only
- Never commit `.env` files  
- Use `envoak` for encrypted storage
